Landing Page Project

This is a simple and responsive Landing Page created using HTML and CSS as part of my web development learning journey.

---

 Project Overview

The goal of this project is to design a clean and user-friendly landing page by applying basic HTML and CSS concepts such as layout, alignment, colors, and responsiveness.

---

 Technologies Used

- HTML5
- CSS3

---

 Features

-  Responsive design (works on different screen sizes)
-  Clean and modern user interface
-  Navigation bar with smooth scrolling
-  Hero section with call-to-action button
-  Features section (Easy Design, Responsive, Fast)
-  Contact section with email link
-  Footer with copyright
